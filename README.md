# Future City China

Static English website for international teams joining the 2026 Future City China Summer Showcase in Hangzhou.

## Local Preview

```bash
python3 -m http.server 4173
```

Open `http://localhost:4173`.

## Deployment

The site is static and deploys from the repository root on Vercel. The production domain is:

```text
futurecity.itccc.app
```
